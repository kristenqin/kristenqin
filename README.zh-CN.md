## 我在做什么

### AI x 学习产品
- [english-practice-app](https://github.com/kristenqin/english-practice-app) - 一个以输出为核心的英语学习产品，重点放在真实表达、反馈闭环和习惯建立上。
- `adaptive-focus-system` - 一个围绕真实日常行为设计的专注支持系统，而不是理想化的效率模板。

### 设计 x 工作流
- [figma-flow-tools](https://github.com/kristenqin/figma-flow-tools) - 围绕 Figma 快捷键、属性学习、token 提取和设计工作流优化的一组工具。

### 开发者工具 x 过程管理
- [github-repo-organizer](https://github.com/kristenqin/github-repo-organizer) - 用来把分散在本地和 GitHub 上的仓库整理成可审阅、可行动清单的工具。
- [git-performance-tool](https://github.com/kristenqin/git-performance-tool) - 把原始 Git 历史转成月度总结、进展快照和可读报告的工具。

### 实用工具 x 自动化
- [data-to-pdfprint](https://github.com/kristenqin/data-to-pdfprint) - 一个把 Excel 数据转换成可直接生产使用的 PDF 标签的 Python GUI/CLI 工具。
- [keyword-highlighter](https://github.com/kristenqin/keyword-highlighter) - 一个面向中文网页阅读效率的规则型关键词高亮 Chrome 插件。
